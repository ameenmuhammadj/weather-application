# Weather App 🌦️
Real-time weather data using OpenWeatherMap API.

## Features
- Current temperature, humidity, wind speed
- Feels like temperature and min/max
- Weather tips based on conditions
- Error handling for invalid cities and no internet

## Requirements
- Python 3
- requests library
- python-dotenv library

## Setup
1. Clone this repository
2. Install dependencies:
   pip install requests python-dotenv
3. Create a .env file in the same folder:
   API_KEY=your_openweathermap_api_key_here
4. Get your free API key from https://openweathermap.org

## Run
python weather_app.py
